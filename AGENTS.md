# Backend instructions

- Repository ini berisi backend NestJS, Prisma, dan PostgreSQL untuk IVORY.
- Frontend Next.js berada di repository `git@github.com:ulumudinsalsabila/onlineshop.git`.
- Pertahankan kontrak response `{ success, data, error }` dan lakukan authorization pada setiap endpoint yang dilindungi.
- Jalankan lint, type-check, dan build sebelum menyerahkan perubahan.
